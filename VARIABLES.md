# Variables in the Restructured ICOS/FLUXNET NetCDF4 File

Generated by `fluxnet_restructure.py` from FLUXNET/ICOS CSV products.
All processing follows the **ONEFlux pipeline** described in:

> Pastorello, G. et al. (2020). *The FLUXNET2015 dataset and the ONEFlux processing pipeline for eddy covariance data.* Scientific Data **7**, 225. [https://doi.org/10.1038/s41597-020-0534-3](https://doi.org/10.1038/s41597-020-0534-3)

---

## File structure

The file contains five groups, each with its own `time` dimension:

| Group | Resolution | Content |
|---|---|---|
| `/` (root) | Half-hourly (HH) | Merged met forcing, NEE, GPP, RECO, LE, H, soil |
| `/fluxnet_dd` | Daily (DD) | Same variables aggregated to daily |
| `/fluxnet_ww` | Weekly (WW) | Weekly aggregations |
| `/fluxnet_mm` | Monthly (MM) | Monthly means |
| `/fluxnet_yy` | Annual (YY) | Annual sums / means |

## Coordinate dimensions

Multi-dimensional flux variables use named coordinate axes instead of the flat per-column naming scheme used in the original FLUXNET CSV files.

### `ustar_threshold` — USTAR filtering variant

Values: `["CUT", "VUT"]`

The friction-velocity (u★) threshold is estimated by bootstrapping two detection methods (change-point CP and moving-point MP) over 100 resampled versions of each year's nighttime data, yielding up to 200 threshold candidates per year. From these, 40 representative percentiles are extracted and used to filter NEE. ([§ USTAR threshold estimation, Pastorello et al. 2020](https://doi.org/10.1038/s41597-020-0534-3))

| Index | Label | Description |
|---|---|---|
| 0 | `CUT` | **Constant USTAR Threshold** — single threshold pooled across all years |
| 1 | `VUT` | **Variable USTAR Threshold** — year-specific, influenced by neighbouring years |

### `nee_variant` — NEE ensemble variant

Values: `["REF", "USTAR50", "MEAN", "SE", "05", "16", "25", "50", "75", "84", "95"]`

From 40 USTAR-filtered NEE realisations, the following summary versions are derived. ([§ Selection of reference NEE variables, Pastorello et al. 2020](https://doi.org/10.1038/s41597-020-0534-3))

| Index | Label | Description |
|---|---|---|
| 0 | `REF` | **Reference** — the realisation most similar to all 40 others (max Nash–Sutcliffe efficiency sum). **Recommended for analysis.** |
| 1 | `USTAR50` | Filtered with the median (50th percentile) USTAR threshold; stable across temporal aggregations |
| 2 | `MEAN` | Arithmetic mean of all 40 realisations |
| 3 | `SE` | Standard error across the 40 realisations |
| 4–10 | `05`…`95` | Distribution percentiles characterising USTAR-threshold uncertainty |

### `partition_method` — CO₂ flux partitioning method

Values: `["NT", "DT"]`

NEE is partitioned into GPP and RECO by two independent methods. ([§ CO₂ flux partitioning, Pastorello et al. 2020](https://doi.org/10.1038/s41597-020-0534-3))

| Index | Label | Description |
|---|---|---|
| 0 | `NT` | **Nighttime method** — respiration–temperature model fitted to nighttime data (Reichstein et al. 2005) |
| 1 | `DT` | **Daytime method** — light-response curve + respiration–temperature model fitted to daytime data (Lasslop et al. 2010) |

### `corr_pct` — Energy-balance correction percentile

Values: `["p25", "p50", "p75"]`

LE and H are corrected for energy-balance non-closure by multiplying by a correction factor (EBC_CF) derived from a ±15-day sliding window. Three percentiles capture the uncertainty in EBC_CF. ([§ Energy and water products, Pastorello et al. 2020](https://doi.org/10.1038/s41597-020-0534-3))

| Index | Label | Description |
|---|---|---|
| 0 | `p25` | 25th percentile of EBC_CF — lower uncertainty bound |
| 1 | `p50` | Median EBC_CF — **recommended for analysis** |
| 2 | `p75` | 75th percentile of EBC_CF — upper uncertainty bound |

### `soil_layer` — Soil profile layer index

Integer 1-D coordinate (1 = shallowest). Layer depths are in site BADM metadata (`VAR_INFO_HEIGHT`).

## Carbon flux variables

### `NEE` — Net Ecosystem Exchange
`(time, ustar_threshold, nee_variant)` · µmol CO₂ m⁻² s⁻¹

Gap-filled net CO₂ exchange between ecosystem and atmosphere. Negative = carbon uptake (sink). After QA/QC and USTAR filtering, remaining gaps are filled by the **Marginal Distribution Sampling (MDS)** method using SW_IN, TA and VPD as drivers. ([§ Gap-filling of NEE, Pastorello et al. 2020](https://doi.org/10.1038/s41597-020-0534-3))

### `NEE_QC` — NEE gap-fill quality flag
`(time, ustar_threshold, nee_variant)` · unitless (0–3)

MDS gap-fill confidence: `0` = measured, `1` = high quality fill, `2` = medium quality fill, `3` = low quality fill.

### `NEE_RANDUNC` — NEE random uncertainty
`(time, ustar_threshold, nee_variant)` · µmol CO₂ m⁻² s⁻¹

Estimated by the Hollinger & Richardson (2005) method using the standard deviation of measured fluxes within a ±7-day, ±1-hour sliding window. ([§ Random uncertainty for NEE, Pastorello et al. 2020](https://doi.org/10.1038/s41597-020-0534-3))

### `NEE_JOINTUNC` — NEE joint uncertainty
`(time, ustar_threshold, nee_variant)` · µmol CO₂ m⁻² s⁻¹

Quadratic combination of USTAR-threshold spread (from the 16th and 84th percentiles, equivalent to ±1 SD) and random uncertainty:

```
JOINTUNC = sqrt( RANDUNC² + ((p84 - p16) / 2)² )
```

([§ Random uncertainty for NEE, Eq. 5, Pastorello et al. 2020](https://doi.org/10.1038/s41597-020-0534-3))

### `NEE_DAY_RANDUNC` / `NEE_NIGHT_RANDUNC`
`(time, ustar_threshold, nee_variant)` · µmol CO₂ m⁻² s⁻¹ · *aggregated groups only*

Daytime and nighttime components of random uncertainty. Only `REF` and `USTAR50` variants are populated in DD/MM/YY groups.

---

### `GPP` — Gross Primary Production
`(time, partition_method, ustar_threshold, nee_variant)` · µmol CO₂ m⁻² s⁻¹

Photosynthetic CO₂ uptake estimated by flux partitioning:
- **NT** (nighttime method): GPP = RECO − NEE, where RECO is from a respiration–temperature model fitted to nighttime data. ([Reichstein et al. 2005](https://doi.org/10.1111/j.1365-2486.2005.001002.x))
- **DT** (daytime method): GPP from a light-response curve model simultaneously fitted to daytime NEE. ([Lasslop et al. 2010](https://doi.org/10.1111/j.1365-2486.2009.02041.x))

([§ CO₂ flux partitioning, Pastorello et al. 2020](https://doi.org/10.1038/s41597-020-0534-3))

### `GPP_QC` — GPP quality flag
`(time, partition_method, ustar_threshold, nee_variant)` · unitless (0–3)

Inherited from the NEE_QC of the NEE used for partitioning.

---

### `RECO` — Ecosystem Respiration
`(time, partition_method, ustar_threshold, nee_variant)` · µmol CO₂ m⁻² s⁻¹

Total ecosystem CO₂ efflux estimated by the same two partitioning methods as GPP. The 40 GPP/RECO realisations propagate the full USTAR-threshold uncertainty from NEE. ([§ CO₂ flux partitioning, Pastorello et al. 2020](https://doi.org/10.1038/s41597-020-0534-3))

### `RECO_QC` — RECO quality flag
`(time, partition_method, ustar_threshold, nee_variant)` · unitless (0–3)

---

## Energy flux variables

### `LE_CORR` — Latent heat flux, EBC-corrected
`(time, corr_pct)` · W m⁻²

Latent heat flux first gap-filled by MDS, then multiplied by the energy-balance closure correction factor (EBC_CF) to enforce energy balance closure (Bowen-ratio assumption). The three percentile variants (`p25`, `p50`, `p75`) reflect uncertainty in EBC_CF. Use `p50` (index 1) for analysis. ([§ Energy and water products, Pastorello et al. 2020](https://doi.org/10.1038/s41597-020-0534-3))

### `H_CORR` — Sensible heat flux, EBC-corrected
`(time, corr_pct)` · W m⁻²

As `LE_CORR`. Correction applied under the assumption that the Bowen ratio (H/LE) is correctly measured.

### `LE_F_MDS` — Latent heat flux, MDS gap-filled only
`(time,)` · W m⁻² · *1-D*

MDS-filled LE without energy-balance correction.

### `H_F_MDS` — Sensible heat flux, MDS gap-filled only
`(time,)` · W m⁻² · *1-D*

## Soil variables

### `TS` — Soil temperature
`(time, soil_layer)` · °C

MDS gap-filled soil temperature at each instrumented depth. Layer index 1 is shallowest. Original columns: `TS_F_MDS_1`, `TS_F_MDS_2`, … ([§ Meteorological products, Pastorello et al. 2020](https://doi.org/10.1038/s41597-020-0534-3))

### `TS_QC` — Soil temperature quality flag
`(time, soil_layer)` · unitless (0–3)

### `SWC` — Soil water content
`(time, soil_layer)` · %

Volumetric soil water content at each instrumented depth. Original columns: `SWC_F_MDS_1`, `SWC_F_MDS_2`, … Range 0–100%.

### `SWC_QC` — Soil water content quality flag
`(time, soil_layer)` · unitless (0–3)

## Meteorological variables (1-D)

All are half-hourly or temporally aggregated scalars. Gap-filling notation:

| Suffix | Method | Description |
|---|---|---|
| *(none)* | measured | Original site measurement |
| `_F_MDS` | MDS | Gap-filled by Marginal Distribution Sampling ([Reichstein et al. 2005](https://doi.org/10.1111/j.1365-2486.2005.001002.x)) |
| `_F_ERA` | ERA-Interim | Gap-filled by ERA-Interim downscaling ([Vuichard & Papale 2015](https://doi.org/10.5194/essd-7-157-2015)) |
| `_F` | Combined | High-quality MDS fill preferred; ERA-Interim used when MDS QC ≥ 2 |
| `_F_QC` | — | Final quality flag: 0 = measured, 1 = MDS fill, 2 = ERA-I fill |

([§ Meteorological products, Pastorello et al. 2020](https://doi.org/10.1038/s41597-020-0534-3))

| Variable | Units | Description |
|---|---|---|
| `TA_F` | °C | Air temperature (combined gap-filled) |
| `TA_F_MDS` | °C | Air temperature (MDS gap-filled) |
| `TA_F_ERA` | °C | Air temperature (ERA-I gap-filled) |
| `SW_IN_F` | W m⁻² | Incoming shortwave radiation (combined) |
| `SW_IN_F_MDS` | W m⁻² | Incoming shortwave radiation (MDS) |
| `SW_IN_POT` | W m⁻² | Theoretical top-of-atmosphere potential SW radiation |
| `LW_IN_F` | W m⁻² | Incoming longwave radiation (combined) |
| `LW_IN_F_MDS` | W m⁻² | Incoming longwave radiation (MDS) |
| `VPD_F` | hPa | Vapour pressure deficit (combined) |
| `VPD_F_MDS` | hPa | Vapour pressure deficit (MDS) |
| `PA_F` | kPa | Atmospheric pressure |
| `P_F` | mm | Precipitation (combined; summed at DD/MM/YY) |
| `WS_F` | m s⁻¹ | Horizontal wind speed |
| `USTAR` | m s⁻¹ | Friction velocity (not gap-filled; used for QC filtering) |
| `NETRAD` | W m⁻² | Net radiation |
| `PPFD_IN` | µmol photon m⁻² s⁻¹ | Incoming photosynthetic photon flux density |
| `CO2_F_MDS` | µmol CO₂ mol⁻¹ | Ambient CO₂ mole fraction (MDS gap-filled) |
| `RH` | % | Relative humidity |
| `G_F_MDS` | W m⁻² | Ground heat flux (MDS gap-filled) |

## Quality-flag conventions

MDS gap-fill flags (`_F_MDS_QC`, `_F_QC`) follow a 0–3 scale:

| Value | Meaning |
|---|---|
| 0 | Measured (no gap-fill applied) |
| 1 | Filled with high confidence |
| 2 | Filled with medium confidence |
| 3 | Filled with low confidence |

For the combined `_F` products, the flag value is 0 (measured), 1 (MDS high-quality fill), or 2 (ERA-Interim downscaled fill). It is **strongly recommended** to filter by QC flag when using gap-filled data. ([§ Using the QC flags, Pastorello et al. 2020](https://doi.org/10.1038/s41597-020-0534-3))

## Recommended variable selections for analysis

| Use case | Recommended selection |
|---|---|
| NEE time series | `NEE[:, VUT=1, REF=0]` |
| NEE uncertainty envelope | `NEE_JOINTUNC[:, VUT=1, REF=0]` |
| GPP (standard) | `GPP[:, NT=0, VUT=1, REF=0]` |
| GPP partitioning comparison | Both NT and DT with VUT REF |
| Latent heat | `LE_CORR[:, p50=1]` |
| Sensible heat | `H_CORR[:, p50=1]` |
| Soil temperature profile | `TS[:, :]` — all layers |
| Met forcing | `TA_F`, `SW_IN_F`, `VPD_F`, `P_F` |
